############################################
##############  dieselTheme  ###############
############################################
- Keycloak custom theme for DieselMedaille.

- Utilization:
    * move dieselTheme directory to keycloak-5.0.0\themes
    * go to keycloak admin console
    * select preferred realm
    * go to Realm Settings -> Themes
    * select dieselTheme from the dropdown menu
    * press Save button
    * enjoy
